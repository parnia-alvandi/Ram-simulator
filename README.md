# RAM Simulator

این پروژه یک شبیه‌ساز حافظه (RAM) و الگوریتم‌های جایگزینی صفحه (Page Replacement Algorithms) است.

## ویژگی‌ها
- پشتیبانی از الگوریتم‌های:
  - FIFO
  - LRU
  - Optimal
  - Clock
- مدیریت جدول صفحات و فریم‌ها
- قابل توسعه و اضافه کردن الگوریتم‌های جدید

## ساخت و اجرا

1. ابتدا مطمئن شوید **CMake** و **GCC** نصب هستند.
2. ساخت پروژه:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
